## How to Resize Your Screenshots

Your current screenshots are 1558 x 452 pixels and need to be resized to 2000 x 1250.

**Option 1: Using macOS sips (fastest)**
```bash
cd metadata
sips -z 1250 2000 withings-sync-1.png
sips -z 1250 2000 withings-sync-2.png
sips -z 1250 2000 withings-sync-3.png
```

**Option 2: Using macOS Preview**
